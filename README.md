# AI-Powered Paraphrasing Tool
---
---

## Project Overview
This project is a console-based AI-powered paraphrasing tool built using 
Transformer models from Hugging Face.

The tool:
- Takes input text
- Generates a paraphrased version
- Preserves original meaning
- Improves clarity and fluency
- Evaluates similarity using ROUGE and Semantic Similarity Score

---

## Technologies Used

- Python
- Hugging Face Transformers
- PyTorch
- NLTK
- ROUGE Score
- Sentence Transformers
- LanguageTool (Grammar Check)

---

## Model Used

- `t5_paraphraser` (Fine-tuned T5 model)

---

## Evaluation Metrics
- ROUGE-1
- ROUGE-L
- Semantic Similarity Score
- Grammar Check
